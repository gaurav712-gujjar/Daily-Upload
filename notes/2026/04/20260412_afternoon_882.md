## 🛠️ What I Built / Debugged

Been working on building a trading bot signal system, focusing on creating a robust indicator that can accurately predict stock price movements based on historical data. The goal is to integrate this indicator with a larger trading bot framework, allowing for automated buy and sell signals. To start, I'm concentrating on developing a simple moving average crossover strategy that can be used as a baseline for more complex indicators.

```
import pandas as pd
import numpy as np

def calculate_signals(data, short_window, long_window):
    signals = pd.DataFrame(index=data.index)
    signals['short_ma'] = data['Close'].rolling(window=short_window).mean()
    signals['long_ma'] = data['Close'].rolling(window=long_window).mean()
    signals['signal'] = 0.0
    signals.loc[(signals['short_ma'] > signals['long_ma']), 'signal'] = 1.0
    signals.loc[(signals['short_ma'] < signals['long_ma']), 'signal'] = -1.0
    return signals

data = pd.read_csv('stock_data.csv')
signals = calculate_signals(data, 20, 50)
print(signals)
```

As I began implementing the moving average crossover strategy, I initially encountered issues with the rolling mean calculation, which was causing the short and long moving averages to be misaligned. I fixed this by ensuring that the rolling window sizes were correctly applied to the historical stock data. The initial results looked promising, but I need to refine the signal generation logic to better handle cases where the short and long moving averages are equal.

I still need to integrate the signal system with a trading bot framework to test its performance in a live trading environment.
