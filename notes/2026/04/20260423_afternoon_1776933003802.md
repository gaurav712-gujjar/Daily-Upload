## 🛠️ What I Built / Debugged

Been working on building a trading bot signal system, specifically designing a simple moving average crossover strategy to generate buy and sell signals. This system will help me identify potential trading opportunities based on the intersection of two moving averages with different time periods. The goal is to create a reliable and efficient system that can be integrated into a larger trading platform.

```
import pandas as pd
import numpy as np

def calculate_moving_averages(data, short_window, long_window):
    data['short_ma'] = data['close'].rolling(window=short_window).mean()
    data['long_ma'] = data['close'].rolling(window=long_window).mean()
    return data

def generate_signals(data):
    signals = pd.DataFrame(index=data.index)
    signals['buy'] = (data['short_ma'] > data['long_ma']) & (data['short_ma'].shift(1) <= data['long_ma'].shift(1))
    signals['sell'] = (data['short_ma'] < data['long_ma']) & (data['short_ma'].shift(1) >= data['long_ma'].shift(1))
    return signals

data = pd.read_csv('stock_data.csv')
data = calculate_moving_averages(data, 50, 200)
signals = generate_signals(data)
```

When I first implemented the strategy, I encountered an issue with the moving average calculations, where the resulting values were not accurate due to missing data points. I fixed this by using the `rolling` function from pandas with the correct window sizes and also handled the missing values properly. Additionally, I had to adjust the signal generation logic to account for the shifting of the moving averages, which was causing incorrect buy and sell signals.
