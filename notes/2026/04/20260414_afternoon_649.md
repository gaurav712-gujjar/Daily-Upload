## 🛠️ What I Built / Debugged

Been working on building a trading bot signal system, focusing on creating a robust and efficient way to generate buy and sell signals based on technical indicators. The goal is to integrate this system with a larger trading platform, allowing for automated trading decisions. I've been exploring different indicators and signal generation techniques to achieve this.

```
import pandas as pd
import numpy as np

def generate_signals(data):
    data['short_ma'] = data['close'].rolling(window=20).mean()
    data['long_ma'] = data['close'].rolling(window=50).mean()
    data['signal'] = np.where(data['short_ma'] > data['long_ma'], 1, 0)
    return data

def calculate_position(data):
    data['position'] = data['signal'].diff()
    return data

# Example usage
data = pd.read_csv('stock_data.csv')
data = generate_signals(data)
data = calculate_position(data)
print(data.head())
```

As I started testing the signal generation function, I realized that the short and long moving averages were not being calculated correctly, resulting in incorrect signals. I fixed this by adjusting the rolling window sizes and ensuring that the data was properly aligned before calculating the moving averages. I also had to modify the position calculation to correctly reflect the changes in the signal.

I still need to integrate the signal system with the trading platform's API to automate the trading process.
