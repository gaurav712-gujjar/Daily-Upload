## 💡 What I Learned Today

Today I learned about scikit-learn Pipeline with ColumnTransformer, which allows me to chain multiple data preprocessing steps together and apply them to specific columns of my dataset. This is particularly useful when working with datasets that have a mix of numerical and categorical features, as it enables me to preprocess each type of feature separately. By using Pipeline and ColumnTransformer, I can simplify my code and improve the readability of my data preprocessing workflow.

```
from sklearn.pipeline import Pipeline
from sklearn.compose import ColumnTransformer
from sklearn.preprocessing import OneHotEncoder, StandardScaler
import pandas as pd

# create a sample dataframe
data = {'category': ['A', 'B', 'A'], 'num1': [1, 2, 3], 'num2': [4, 5, 6]}
df = pd.DataFrame(data)

# define preprocessing steps
numeric_features = ['num1', 'num2']
categorical_features = ['category']

numeric_transformer = StandardScaler()
categorical_transformer = OneHotEncoder()

preprocessor = ColumnTransformer(
    transformers=[
        ('num', numeric_transformer, numeric_features),
        ('cat', categorical_transformer, categorical_features)
    ]
)

pipeline = Pipeline(steps=[('preprocessor', preprocessor)])

# fit and transform the data
df_transformed = pipeline.fit_transform(df)
```

What surprised me is how easy it is to define and apply multiple preprocessing steps to different columns of my dataset using ColumnTransformer. I also need to watch out for the order in which I define the preprocessing steps, as this can affect the final output of my pipeline. Another important consideration is the handling of missing values, which can be tricky when working with pipelines.
