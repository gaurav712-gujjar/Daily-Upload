## 🛠️ What I Built / Debugged

Been working on building a trading bot signal system, specifically focusing on implementing a moving average crossover strategy to generate buy and sell signals. The goal is to create a system that can be easily integrated with various trading platforms and can handle multiple symbols and timeframes. I started by defining the strategy's parameters, such as the short and long window lengths, and the threshold for generating signals.

```
import pandas as pd
import numpy as np

class MovingAverageCrossover:
    def __init__(self, short_window, long_window, threshold):
        self.short_window = short_window
        self.long_window = long_window
        self.threshold = threshold

    def calculate_signals(self, data):
        data['short_ma'] = data['close'].rolling(self.short_window).mean()
        data['long_ma'] = data['close'].rolling(self.long_window).mean()
        data['signal'] = np.where(data['short_ma'] > data['long_ma'] + self.threshold, 1, 0)
        data['signal'] = np.where(data['short_ma'] < data['long_ma'] - self.threshold, -1, data['signal'])
        return data

# example usage
data = pd.read_csv('stock_data.csv')
strategy = MovingAverageCrossover(20, 50, 0.5)
data = strategy.calculate_signals(data)
```

Initially, I encountered issues with the rolling window calculations, which resulted in NaN values for the first few rows of the dataframe, but I fixed this by using the `min_periods` argument in the `rolling` function. I also had to adjust the threshold value to avoid excessive signal generation, and I added a small buffer to the signal calculation to prevent unnecessary buy and sell signals. 

What's left to do is to integrate this strategy with a backtesting framework to evaluate its performance on historical data and optimize the parameters for better results.
