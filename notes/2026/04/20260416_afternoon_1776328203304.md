## 🛠️ What I Built / Debugged

Been working on building a trading bot signal system, specifically designing a function to generate buy and sell signals based on a given set of indicators. I'm using a combination of moving averages and relative strength index (RSI) to determine when to enter or exit a trade. My goal is to create a system that can be easily integrated with popular trading platforms.

```
def generate_signals(prices, short_ma, long_ma, rsi_period):
    signals = []
    for i in range(len(prices)):
        if i > long_ma:
            short_ma_value = sum(prices[i-short_ma:i]) / short_ma
            long_ma_value = sum(prices[i-long_ma:i]) / long_ma
            rsi = calculate_rsi(prices, rsi_period, i)
            if short_ma_value > long_ma_value and rsi < 70:
                signals.append(1)  # buy signal
            elif short_ma_value < long_ma_value and rsi > 30:
                signals.append(-1)  # sell signal
            else:
                signals.append(0)  # neutral signal
    return signals

def calculate_rsi(prices, period, index):
    gains = []
    losses = []
    for i in range(1, len(prices)):
        if prices[i] > prices[i-1]:
            gains.append(prices[i] - prices[i-1])
            losses.append(0)
        else:
            gains.append(0)
            losses.append(prices[i-1] - prices[i])
    avg_gain = sum(gains[index-period+1:index+1]) / period
    avg_loss = sum(losses[index-period+1:index+1]) / period
    return 100 - (100 / (1 + avg_gain / avg_loss))
```

Initially, my function was not accounting for cases where the short MA was equal to the long MA, which would result in a division by zero error when calculating the RSI. I fixed this by adding a simple check to ensure that the short MA value is not equal to the long MA value before generating a signal. I also had to adjust the RSI calculation to handle cases where the average gain or loss was zero, which would also result in a division by zero error.

I still need to integrate this function with a data feed to test its performance in real-time and adjust the parameters to optimize its accuracy.
