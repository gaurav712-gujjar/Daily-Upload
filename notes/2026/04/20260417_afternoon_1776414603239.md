## 🛠️ What I Built / Debugged

Been working on building a trading bot signal system, specifically designing a simple moving average crossover strategy to generate buy and sell signals. The goal is to create a system that can automatically send signals to a trading platform based on the intersection of short and long term moving averages. I spent most of the day setting up the data feed and calculating the moving averages.

```python
import pandas as pd
import numpy as np

def calculate_moving_averages(data, short_window, long_window):
    data['short_ma'] = data['Close'].rolling(window=short_window).mean()
    data['long_ma'] = data['Close'].rolling(window=long_window).mean()
    return data

def generate_signals(data):
    signals = pd.DataFrame(index=data.index)
    signals['signal'] = 0.0
    signals.loc[(data['short_ma'] > data['long_ma']) & (data['short_ma'].shift(1) <= data['long_ma'].shift(1)), 'signal'] = 1.0
    signals.loc[(data['short_ma'] < data['long_ma']) & (data['short_ma'].shift(1) >= data['long_ma'].shift(1)), 'signal'] = -1.0
    return signals

data = pd.read_csv('stock_data.csv')
data = calculate_moving_averages(data, 50, 200)
signals = generate_signals(data)
```

At first, I encountered issues with the data feed, as the CSV file was not being read correctly, resulting in NaN values in the moving average calculations. I fixed this by checking the data types of the columns and converting the 'Date' column to datetime format, which allowed the rolling function to work correctly. I also had to adjust the signal generation logic to account for cases where the short and long term moving averages were equal.
