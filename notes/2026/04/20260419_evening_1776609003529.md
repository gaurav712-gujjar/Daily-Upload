I realized that a major bottleneck was the repeated calculation of certain indicators, which could be avoided by storing and reusing previously computed values. This optimization is crucial as it saves a substantial amount of computational resources and reduces the overall execution time.

```
def calculate_indicators(data):
  indicators = {}
  for symbol in data:
    if symbol not in indicators:
      indicators[symbol] = calculate_symbol_indicators(data[symbol])
  return indicators

def calculate_symbol_indicators(data):
  # calculate various indicators like moving averages, RSI, etc.
```
