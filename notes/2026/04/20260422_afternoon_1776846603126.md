## 🛠️ What I Built / Debugged

Been working on building a trading bot signal system, specifically designing the logic for generating buy and sell signals based on technical indicators. The goal is to create a robust system that can handle various market conditions and provide accurate signals for trading decisions. I focused on implementing a simple moving average crossover strategy, where a short-term MA crosses above or below a long-term MA to generate a signal.

```
def generate_signal(short_ma, long_ma, current_price):
    if short_ma > long_ma and short_ma[-1] <= long_ma[-1]:
        return "buy"
    elif short_ma < long_ma and short_ma[-1] >= long_ma[-1]:
        return "sell"
    else:
        return "hold"

def calculate_ma(prices, window):
    ma = []
    for i in range(len(prices)):
        if i < window:
            ma.append(None)
        else:
            ma.append(sum(prices[i-window:i]) / window)
    return ma

short_ma = calculate_ma([10, 12, 15, 18, 20], 3)
long_ma = calculate_ma([10, 12, 15, 18, 20], 5)
signal = generate_signal(short_ma, long_ma, 22)
print(signal)
```

Initially, I encountered issues with the MA calculation, as the list indexing was incorrect, resulting in a shift in the MA values. I fixed this by adjusting the loop range in the calculate_ma function to handle the initial None values correctly. The corrected code now generates accurate MA values and corresponding signals.

I still need to integrate the signal generation logic with the trading bot's execution system, which will involve connecting to a brokerage API and placing trades based on the generated signals.
