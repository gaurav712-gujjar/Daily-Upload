## 🛠️ What I Built / Debugged

Been working on building a trading bot signal system that can send out buy and sell signals based on moving averages and other technical indicators. I'm trying to create a system that can be easily extended to include more indicators and strategies in the future. My goal is to have a system that can be backtested and validated on historical data.

```python
import pandas as pd
import numpy as np
from sklearn.metrics import mean_squared_error

def calculate_signals(data):
    short_window = 20
    long_window = 50
    signals = pd.DataFrame(index=data.index)
    signals['signal'] = 0.0
    signals['short_ma'] = data['close'].rolling(window=short_window).mean()
    signals['long_ma'] = data['close'].rolling(window=long_window).mean()
    signals['signal'][short_window:] = np.where(signals['short_ma'][short_window:] > signals['long_ma'][short_window:], 1.0, 0.0)
    return signals

data = pd.read_csv('stock_data.csv')
signals = calculate_signals(data)
print(signals.head())
```

What was tricky at first was getting the rolling averages to line up correctly with the rest of the data, but I fixed it by making sure to use the same index for the signals DataFrame as the original data. I also ran into an issue with NaN values in the rolling average calculations, which I resolved by using the `min_periods` argument of the `rolling` function. 

I still need to backtest the system on a larger dataset to see how it performs in different market conditions.
